# Google Sheets Appender

## Setup

```shell
 # install pyenv on macOS
 brew install pyenv
```

```shell
 # install Python version using pyenv
 pyenv install 3.13.1
```

```shell
 # set Python version using pyenv
 pyenv global python3.13.1
```

```shell
 # install Python libraries
 pip3.13 install google-auth-oauthlib google-api-python-client
```

```shell
 # run the scraper
 python3.13 appender.py
```
